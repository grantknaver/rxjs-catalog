Kinds of Operators

Conditional

For use-cases that depend on a specific condition to be met.

Creation

These operators allow the creation of an observable from nearly anything.

Join

Combines items emitted by two Observables whenever an item from one Observable is emitted during a time window defined according to an item emitted by the other Observable

Transformation

Transforming values as they pass through the operator chain is a common task.

Filtering

These operators provide techniques for accepting values from an observable source and dealing with backpressure.

Utility

From logging, handling notifications, to setting up schedulers, these operators provide helpful utilities
